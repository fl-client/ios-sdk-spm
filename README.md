### Setup
To Configure Swift Package Manager to resolve Quickplay libraries for iOS and tvOS

#### Prerequisites

Create a `.netrc` file under your home directory with your JFrog credentials. 
 
    ```
    machine firstlight.jfrog.io
    login <USERNAME>
    password <PASSWORD>
    ```
